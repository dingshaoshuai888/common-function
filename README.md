# base
android 常用的一些工具类，依赖 base 实现相关的接口

# Quick Setup

Step 1. Add the JitPack repository to your build file
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
Step 2. Add the dependency
```
dependencies {
    implementation 'com.github.dingshaoshuai888:common-function:1.0.0'
}
```
